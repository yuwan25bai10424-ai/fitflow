## 📌 Overview
Gym AI is a Python-based CLI application that allows users to:
- Sign up and log in
- Generate AI-based fitness plans
- View and manage saved fitness data

The system uses:
- Machine Learning models (Linear Regression & Decision Tree)
- JSON file storage for user data
